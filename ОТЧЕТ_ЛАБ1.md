## Жизненный цикл файлов в Git

В процессе работы файлы проходят через следующие состояния:

1. **Untracked (неотслеживаемый)** - новый файл, Git о нем не знает
2. **Staged (в индексе)** - файл добавлен командой `git add`, готов к коммиту
3. **Committed (зафиксирован)** - файл сохранен в репозитории командой `git commit`
4. **Modified (изменен)** - файл изменен в рабочей директории, но не добавлен в индекс

Эти состояния были продемонстрированы в ходе выполнения лабораторной работы.
